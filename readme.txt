An easy way to Create Pinterest Pinboard Page on your WordPress website as a shortcode.

== Description ==

This is an early version of the Create Pinterest Pinboard Page for WordPress, please let me know at http://akashsoni.com if you have suggestion how to make it better.

The plugin Create Pinterest Pinboard Page your pintrest pins on WordPress site.

	1 ) You need to enter your pintrest username and boradname in Shortcode from. if you not enter any boardname then it show latest pins on page. 

	2 ) you can set color of Pin box background,Pin box text , Show more button text and Show more button background.

	3 ) limitation of pins.

	4 ) Set 10 different style of pinbox.

	5 ) Set row #number of columns.

This plugin feched maximum 50 pins from your pintrest account.

This plugin is maintained by Akash soni, http://akashsoni.com 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/as_pinterest_page` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go To admin side AS Pinterest Page Settings " Create New Pinterest Page " Enter all detail and save that
4. Copy shortcode and add in page content box after then save the page. Click view page.
